> 防御型利润表主要揭示公司对于外部资金投入的依赖程度，侧重对公司破产风险的提示，适合于那些厌恶风险的保守型投资者；
> 激进型利润表侧重于对公司资本报酬率的评价，包括股东权益回报率，适合于激进型投资者，如风险投资者。

## 一、防御型利润表

简单来说，防御型利润表，就是把分摊费用一次性扣除。要特别注意那些资本支出远大于折旧的企业，很可能在粉饰报表。

判断一家公司是否具有自给自足的持续发展能力，更简单的就是用经营性现金流量减去公司的资本性支出和企业并购支出。这三项数据在现金流量表中都能找到，股票交易软件中F10里也都有。

总结：只需要调整固定资产投资和营运资产投资，并减掉投资收益即可。即用资本支出，减去折旧，得到固定资产投资。用今年的营运资本（营收账款-应付账款）减去去年的营运资本，得到净营运资本。即利润里面减掉固定资产投资、营运资产投资和投资收益，即可得到防御型利润表。


## 二、激进型利润表

激进型投资者的主要目标，就是投资那些体制健全且较行业平均收益更具吸引力的公司。

激进型利润表就是将企业的研发支出和广告投入从企业的经营费用中调整出来，将其资本化为企业的资产。进一步讲，通常是将资本化的无形资产净值当作企业的一项具体资产，就如同借款、所有者权益资金等一样。在其他条件相同的情况下，企业每赚取1元利润所耗用的资本越少越好，就如同我们在获得相同薪水的前提下，每周的工作时间越短越好。即资产收益率指标，也就是我们常说的ROE，这才是获得投资回报的依据。

更简单的方法就是，从利润表中找出息税前利润（EBIT）的数据，然后除以公司负债和股东权益之和，即总资产，就可以得到公司的税前资本收益率。如果该收益率大于18%，意味着公司可能会有激进型的利润，如果大于30%，投资之后基本上问题就不大。

对于财报不是很清楚的投资者，只需要抓住几点即可：
1. 净资产收益率ROE，超过15%；
2. 毛利率超过40%；
3. 净利率超过5%；
4. 息税前资本收益率大于18%；
5. 这些指标至少持续了3年。

总结：用息税前利润（EBIT）除以总资产（股东权益加负债），得到息税前资产收益率，如果高于18%，激进型利润大概就是正的。


## 三、盈利能力图

简单来说，就是防御型利润和激进型利润都大于零的企业就比较保险了，即无论利润表怎么调整，都不会出现亏损，最差也就是能自负盈亏。如果利润越来越多，说明公司正在成长，市场正在开拓，也就有长期投资的基础。所以能长期进行价值投资的公司，就是能不断创造收入的公司。

而那些只要把投资算到费用里，就立马亏损的公司，和那些一次性支出费用过多，造成后面几年都盈利的公司，都是不具备投资价值的。

只找那些利润，以及调整后保守型利润和激进型利润，都为正值的公司。这样的公司才具备长期投资的基础。


## 四、典型的盈利能力

典型的盈利能力，即好的成长股一般都具备的盈利能力。就是无论用防御利润表调整，还是用激进型利润表调整，即无论是把投资一把计入费用，还是把无形资产扣除之后，这家公司都是盈利的。

那么作为一个谨慎而又贪婪的长期投资者，最安全的就是持续关注或投资那些处于盈利能力区域的公司。就盈利质量而言，处于这一区域的公司一般是既能在经营资金上自给自足，又能够创造价值的公司。


## 五、最理想的盈利能力图形：阶梯式递增图形

具有持续的盈利增长能力的公司，是那些处于盈利能力图中的盈利能力区域，并且经营业绩点每年都向右上方向移动的公司，这不仅是因为它们盈利规模日益增大，还因为它们即使是按照防御型模式和激进型模式的要求进行衡量，业绩也依旧表现良好。只要你能给这些阶梯递增型盈利公司假以时日，它们定能为你带来巨额的资本投资利得收益。

在界定一家公司是否为阶梯递增型盈利公司之前，究竟需要观察多少年，作者认为能有`两年`的经营历史业绩就基本可以了。毕竟，我们的目标是在这类潜力公司刚起步时就介入并买入它们，如果五年之后再投资，其公司的市盈率可能早就被其他看好该公司的投资者抬高了。


## 六、没有一成不变的股票

### 1. 首先，对于大多数公司而言，盈利质量都会逐年产生变化。

这主要是受到各种因素包括：新竞争对手的闯入、市场饱和程度、客户消费需求的改变以及关键人员的流失等影响所致。一定要“持续关注你所投资的公司”​。

### 2. 其次，尽管那些阶梯递增型盈利公司通常都会为投资者带来大量的资本利得，但我们依然要密切关注企业价值的变化。

在你投入大笔资金时，应该知道“即使是大型优质公司的股票，也完全可能股价表现不佳”​。


## 七、市盈率

将介绍两种评价标准，以便对投资对象迅速作出评判。**一种是基于防御型利润表的债务偿还周期，另一种是基于激进型利润表的新增投资收益率**。这两项指标都是建立在盈利能力评价的方法论基础之上。

### 1. 债务偿还周期

如何才能了解企业对杠杆效用的使用是否过度（即是否存在过度的举债经营）​？一种评价企业信用质量高低的方法是计算企业的**产权比率**，该指标可以反映企业债务规模与企业净资产价值之间的数量对比关系。

但对大多数企业而言，一家公司的固定资产如果放到其他企业或业务上，常常发挥不了多大用处，也就难以用于偿债。

所以衡量企业信誉的一种更理想的指标是**债务偿还周期**。该指标反映的是企业利用所获得的防御型净利润来还债，大概需要多长时间，或者需要几年才能还清？这一指标的评判原则是：债务偿还周期越短，企业的偿债信用就越好。

**建议投资者回避那些债务偿还周期超过5年的企业，因为这些企业风险太大，不值得冒险投资。**

``` bash
“标准化”防御型利润 = 过去4年的防御型净利润加总后再除以4年 = 4年平均值

债务偿还周期 = 债务及其等价物 / 防御型利润
```

**注意：**
> 1. 作为对公司财务安全性的补充衡量指标，计算债务偿还周期时，应采用“标准化”防御型利润来计算企业的债务偿还周期。之所以选择4年进行平均，主要是考虑到有典型波动循环特征的企业，一个循环周期大概是4年时间，因此按照4年计算年均防御型利润来平滑和消除波动循环所产生的影响。
> 2. 企业债务通常包括营运资本信用贷款、长期的流动性贷款、长期借款以及融资租赁等。这些债务金额都可以在资产负债表的负债栏目下找到。至于这里所说的负债等价物，则主要指的是资本化的经营租赁。

### 2. 新增投资收益率

新增投资收益率的计算是用激进型税后经营净利（eNOPAT）每年的变动额除以激进型资本的年变动额。其中激进型税后经营净利等于激进型利润与激进型资本成本之和，换句话说，就是企业没有扣除各项融资的资金成本之前所获得的经营利润。简写中的前缀"e"是“激进”英文的缩写，表示除其他事项之外，企业所拥有的资本化无形资产。

``` bash
激进型税后经营净利（eNOPAT） = 激进型利润 + 激进型资本成本总额

激进型资本的年变动额 = 当年的激进型资本 - 上一年的激进型资本

新增投资收益率 = 激进型税后经营净利（eNOPAT）每年的变动额 / 激进型资本的年变动额
```

具有真正盈利能力的公司往往能够获得较高的年新增投资收益率，而较高的新增投资收益率又代表着公司管理当局合理有效地进行了资本配置与使用。新增投资收益率指标一般不容易被粉饰，不过，如果新增投资收益率指标连年令人失望的话，最终也会影响到企业整体的资本回报水平，并进而影响到企业的激进型利润。

### 3. 总结

购买具有**较短债务偿还周期以及较高新增投资收益率水平**的公司股票，将在股票市场上获得更高的收益。


## 八、竞争优势

在购买公司的股票之前，依然需要考虑以下事项：

> 1. 这家公司的竞争优势究竟有哪些；
> 2. 这些优势估计能够保持多久。

### 1. 这家公司的竞争优势究竟有哪些

以下是企业通常区别于其他竞争对手的几个主要方面，也就是企业可能具有独特竞争优势的几个方面：

1. 拥有世界级品牌（比如可口可乐、耐克、箭牌等）​。
2. 以股东利益为主导的企业经营管理。
3. 员工的文化技能培训。
4. 强大的客户服务体系。
5. 创造力和创新能力。
6. 有价值的专利和商标。
7. 有助于经营收入增长的研发与广告投入。
8. 在顾客方面的大笔（促销）开支卓有成效，形成了企业有竞争力的产品（客户群）​。
9. 具有明显优势的市场占有份额。
10. 有效的利益分配机制。
11. 有利的地理位置。
12. 拥有特许权转让协议。
13. 低廉的产品成本优势。
14. 拥有市场定价权。
15. 在反垄断法律约束之下，难以出现能与之抗衡的竞争对手。

### 2. 这些优势估计能够保持多久

还需要评估一下特定环境中，公司具有的这些竞争优势可以维持多长时间。我们认为，公司竞争优势能够持续的时间越长越好。作为一名投资者，除了需要掌握好各项投资分析技巧之外，还应该能够回答“企业及其竞争优势能够持续多久”之类较难回答的问题。

下面一些问题，不妨自问一下是否都了解：

1. 这家公司的优势和弱势有哪些？
2. 公司主要竞争对手的优势和弱势有哪些？​（需要注意的是，目前在考虑公司竞争对手时，一般也包括公司的供应商和消费客户。​）
3. 公司生产或提供的商品是否能够满足人类的基本需求或企业的日常需要？
4. 公司所生产或提供的商品，成本是否已经控制到最低？能够将成本控制到最低的企业，在经济困难时能够保存实力；而当经济再次繁荣时，又可以利用它们的优势，来提高业绩。
5. 公司是否具有财务实力？根据资产负债表进行分析有助于你将竞争对手抛之脑后。
6. 公司是否拥有主导性或成长性的市场份额？只有投资于那些具有行业主导地位或者拥有足量市场份额的公司，才算得上是正确的投资。
7. 公司所处的行业竞争是否能够促进企业的进一步发展？

### 3. 在分析评价一家公司在市场上的主导地位估计可以维持多久的时候，应该结合企业的运营模式进行综合考虑。

超前技术也许意味着公司在防御型利润和激进型利润两方面都有可能快速增长，因此公司的经营业绩点也就会落在盈利能力图的右上象限。不过，处于图中盈利能力区域的这些公司也会吸引来大量的竞争对手。如同我们前面曾经讲到的那样，这些竞争对手又会带来更新颖、更快捷、更便宜、更有效的产品和服务。如果你是一名顾客，创新对你来讲可能是很奇妙和兴奋的；但如果你是企业主的话，创新可能就是你的敌人，它完全有可能使你的产品或服务一夜之间遭到淘汰。

**最重要的竞争优势应该还是公司的经营管理。**管理者是否具有能力维持公司以往突出的经营业绩，他们对公司各项资产的配置是否合理，管理者的切身利益是否与股东的切身利益合理匹配、有效结合？


## 九、估价

### 1. 估价分析步骤

在考虑一家公司的价值评估时，我们不能仅仅停留在对购买公司的部分股票的价值评估，而是要放大到对获得整个公司全部股份的价值评估！这样就把企业的价值评估放在一个更广阔的背景下进行考虑。下面分为五个步骤来分析：

**1. 明确你希望拥有的总收益。**

假设在评价了公司的盈利质量和竞争优势之后，你希望在未来10年中，每年都能获得15%的总投资回报率，那就是你要求的最低投资收益率（当然，这仅仅是你想要得到15%的收益率，并不代表就一定会发生）​。企业的总收益率一般等于股份的红利收益和资本利得之和，通常会比股票价格略高一点：

``` bash
总收益＝股份红利收益＋资本利得

# 也就意味着：

资本利得＝总收益－红利收益＝15%－1.5%＝13.5%
```

**2. 计算箭牌公司的市场价值。**

``` bash
公司的市场价值 = 流通在外的普通股股数 × 股票单价 + 债务的价值 - 现金及现金等价物
```

**3. 计算公司未来10年的市场价值。**

``` bash
未来10年的市场价值 = 期初市值 * (1 + 增长率)^10
例：
期初：128.58亿
增长率：13.5%
未来10年的市场价值 = 128.58*(1+0.135)^10 = 456.17
```

**4. 估计第10年年末的市盈率（合理估计）​。**

假设到第10年公司的市盈率是17倍（按照合理预期，如果公司10年后依然处于盈利能力区域）​，这意味着第10年的净利润是27亿元（即届时的市场价值456亿元除以17）​。

**5. 最后，日常性测试。**

假如当前公司的净利润是4.02亿元。如果要在第10年达到27亿元的净利润，需要在目前的基础上翻六七番，大约平均每年的利润增长率要达到21%。

公司的年利润增长率是否能够达到这样的水平（即21%）​，在第10年来临之前，我们都无法确认这一点。不过，公司过去的利润增长率为我们的预测提供了参考。假如公司近10年间的年平均利润增长率为10.8%。根据过去的表现，未来10年，21%的利润增长率似乎有些遥不可及了。如果未来10年公司的年利润增长率能够达到15%，那它现在的股价就是被高估的。

### 2. 估价之惑

最好的公司（不一定是表现最好的股票）是那种阶梯递增型盈利公司。这类公司不仅仅是规模越来越大，而且在各个方面都得到了进一步的改善。可以确定的是，这种阶梯递增型盈利公司一定存在一项或多项核心竞争力，你只需要找到这些竞争优势究竟表现在哪些方面，并且分析这些优势估计可以持续多久。

当然，其他投资者也同样想持有这类杰出的优质公司，​（竞争的结果就是）大多数这类阶梯递增型盈利公司的市盈率往往很高。为了平抑这种成长型公司的股价被高估的风险，我们给出以下四条建议：

**1.盈利的成长前景。**应确定这家公司正在构建或完善一套全面而阶梯递增的盈利模式。最好不要投资那些利润质量不高而市盈率却很高的公司。
**2.考虑一下那些小公司。**关注那些尚处于成长期的有良好盈利前景的公司。
**3.平均成本策略。**没有必要一次把钱全部都投出去，取而代之的是，均摊你的成本。
**4.寻找冷门行业中的领军公司。**

### 3. 下列几类公司建议尽可能回避：

1. 首次公开发行股票的公司。
2. 生产流行产品的公司。这些公司产品周期短，热得快、冷得也快。因此其股票价格也可能涨得很快，但跌的时候速度更快。
3. 客户陷入困境的公司。有些麻烦就像鲑鱼产卵一样，逆流而上。
4. “鸡尾酒会上的股票”公司。
5. 客户基础薄弱的公司（营业收入主要来自屈指可数的那么几个客户）。
6. 财务杠杆过高（即资产负债表中显示有过多债务）的公司。
7. 内部知情人大量抛售股票的公司。
8. 连续并购的公司。
9. 反复出现“一次性”费用（即存在过于频繁的非经常性损失）的公司。在过去十年中，有超过两次以上的非经常性和（或）异常重大损失的公司，可能存在着应计制利润被高估或公司高管滥用股东权益资本的行为。此外，有过财务重述（即对以前年度财务报表进行调整）的公司也应当尽量回避。
10. 管理层频繁变动的公司。
11. 企业经营战略明显发生变化的公司。
12. 已经过了全盛时期的古老而著名的公司。这类公司常常只剩下品牌名称的价值在支撑着公司股价，所以常常体现的是两位数的高市盈率和一位数的营业收入增长率之间的极大反差。
13. 商誉占总资产比例较大的公司。
14. 稀释性潜在股份较多的公司。
15. 投资银行家执掌的公司。这些人很少有人拥有经营好一家公司的丰富经验，包括如何研究与开发公司的新产品或新服务、如何降低生产成本、如何延伸拓展销售渠道等。
16. 管理层不称职的公司。

# 总结（利用AI生成的总结）

> 说明：下面的内容是向AI提问该书时生成的内容，有些数字可能是书中例子的计算结果，所以不一定要拿来做为衡量的标准。

## 一、防御型利润表：聚焦“稳定性”

核心目标：识别在行业下行期仍能维持稳定盈利和现金流以及分红能力的公司，这类企业通常具备抗周期性、低杠杆和高客户黏性等特点。
适用企业：现金流稳定、负债低、行业地位稳固的公司（如消费龙头、公用事业）。

### 1. 关键指标筛选

| 指标 | 标准 | 分析要点 | 数据来源 |
| ---- | ---- | -------- | -------- |
| 营业收入波动率 | 过去5年营收增速标准差 ≤ 行业平均值的50%（如消费行业标准差≤8%），且无大幅下滑 |  | 年报、Wind/同花顺行业对比工具 |
| 毛利率稳定性 | 毛利率长期维持在30%以上（制造业可适当放宽），且波动幅度 ≤ 5%（如连续5年毛利率在35%-40%之间）| 显示产品竞争力强，成本控制能力佳 | 利润表“营业成本”科目计算 |
| 经营性现金流/净利润 | 比值持续≥1.2（若<1，需排除应收账款周转天数大幅上升的情况） | 证明盈利质量高，现金创造能力强（避免依赖应收账款周转） | 现金流量表“经营活动现金流净额” |
| 负债率 | 总资产负债率≤50%，流动比率≥2，有息负债（或短期借款+长期借款+应付债券）/总资产 ≤ 30%（金融地产行业需单独分析） | 长期借款占比低，财务结构稳健 | 资产负债表“短期借款+长期借款+债券” |
| 研发费用占比 | 研发投入适中（通常5%-10%） | 聚焦核心业务升级而非盲目创新 |  |
| 分红比例 | 连续3年股息支付率 ≥ 30% | 体现股东回报能力 | 利润表“应付股利”与“归属于母公司净利润” |

### 2. 行业适配性调整

* 消费类行业（如食品饮料）：重点关注品牌溢价和渠道稳定性。
* 公用事业（如水电燃气）：考察特许经营权、政策补贴和成本转嫁能力。
* 银行/保险：分析净息差、不良贷款率和保费收入稳定性。

### 3. 风险提示：

* 检查应收账款增速是否超过营收增速（防御型企业通常以现款销售为主）。
* 剔除商誉占总资产比例超过10%的公司（避免减值风险）。
* 防御型陷阱：低估值但增长停滞（如部分传统零售股）。

### 4. A股实操示例

* 步骤1：从沪深300指数成分股中筛选连续3年ROE≥15%、股息率≥3%的公司。
* 步骤2：剔除商誉占净资产比例超过20%的企业（降低减值风险）。
* 步骤3：验证过去5年经营性现金流是否覆盖投资性现金流（防御型企业通常为正向）。

### 5. A股适用行业
* 必选消费（海天味业、双汇发展）：需求刚性，品牌护城河深。
* 公用事业（长江电力、华能水电）：特许经营+稳定现金流。
* 医药龙头（云南白药、片仔癀）：独家品种+高毛利支撑。


## 二、激进型利润表：聚焦“成长性”

核心逻辑：识别高研发投入、高资本开支、市场快速扩张的企业，容忍短期亏损，但需验证长期增长逻辑。
核心目标：识别高增长潜力（未来3-5年可能爆发式增长）但伴随较高风险的标的，这类企业往往处于成长期，依赖资本投入和市场扩张。
适用企业：高研发投入、高资本开支、市场扩张中的公司（如科技、新能源、创新药）。

### 1. 关键指标筛选

| 指标 | 标准 | 分析要点 | 数据来源 |
| ---- | ---- | -------- | -------- |
| 营收增速 | 连续3年营收增速≥20%，或过去3年复合增长率≥30% | 新兴行业如AI、半导体等可放宽至40%+ | 利润表“营业收入”科目 |
| 研发费用率 | 研发费用占比≥10%，且有专利储备或技术壁垒 | 半导体、创新药企业需≥15% | 利润表“研发费用”科目 |
| 资本开支/营收 | ≥25%（需与产能扩张计划匹配） | 资本开支增速与营收增速匹配，且ROIC（投入资本回报率）≥WACC（加权平均资本成本） | 现金流量表“购建固定资产支出” |
| 有息负债率 | 可接受短期负债率高（如≤60%），但需确保经营性现金流能覆盖利息支出 |  |  |
| 市销率（PS） | 行业平均PS的2倍以内（避免估值泡沫），结合未来3年盈利预期判断合理性 |  |  |
| 客户集中度 | 前五大客户收入占比 ≤ 40% | 过高则依赖大客户风险大 | 年报“前五名客户销售额”披露 |

### 2. 成长性赛道聚焦
* 科技类（如AI、半导体）：关注技术迭代速度和客户集中度。
* 医药生物（创新药/CRO）：分析管线储备和临床试验进度。
* 新能源车：考察产能利用率、毛利率改善趋势和全球化布局。

### 3. 风险提示：
* 现金流风险：
    计算自由现金流（FCF）= 经营性现金流 - 资本开支，若连续2年为负需评估融资能力
* 激进型陷阱：高增长但技术路线错误（如某些光伏技术被颠覆）。

### 4. A股实操示例
* 步骤1：在科创板和创业板中筛选近2年营收增速前50%的公司。
* 步骤2：排除商誉减值风险高的企业（如并购标的占比过高）。
* 步骤3：计算自由现金流（FCF）= 经营活动现金流 - 资本开支，若连续2年为正则进入备选池。

### 5. A股适用行业
* 硬科技（中芯国际、兆易创新）：国产替代+高研发投入。
* 新能源（宁德时代、隆基绿能）：政策驱动+技术迭代。
* 创新药（百济神州、信达生物）：管线估值+临床突破。


## 三、两类利润表的对比与组合策略

| 维度 | 防御型利润表 | 激进型利润表 |
| ---- | ------------ | ------------ |
| 财务特征 | 低负债、高现金流、稳定分红 | 高负债、负自由现金流、股权融资频繁 |
| 估值方法 | 股息贴现模型（DDM）、PE/PB | 市销率（PS）、远期PE、管线折现模型（rNPV） |
| 估值锚点 | 股息率（≥3%）、PE（≤行业中位数） | PS（≤行业2倍）、PEG（≤1） |
| 买入时机 | 行业低谷期（如消费股PE跌至15倍以下） | 技术突破/政策利好初期（如新能源补贴出台） |
| 卖出信号 | 股息率<2%、ROE连续2年下降 | 营收增速跌破20%、研发进度滞后 |
| 仓位配置 | 占总仓位的50%-70%（核心持仓，降低波动） | 占总仓位的20%-30%（卫星持仓，捕捉超额收益） |


## 四、总结：从理论到落地的关键点

1. 防御型利润表的本质是“活下去”，需关注现金流安全垫（货币资金/短期债务 > 1.5）。
    现金安全比率 = 货币资金 / 短期有息负债 ≥ 1.5
2. 激进型利润表的核心是“赌未来”，需紧盯边际变化（如季度订单增速、研发专利获批）。
    - 关键动作：
        - 每季度跟踪订单增速（如宁德时代电池订单）。
        - 监控研发专利数量（如中芯国际28nm以上工艺占比）。
3. 动态调整：每季度财报发布后更新数据，防御型企业允许5%指标波动，激进型企业需严格止损（如营收增速跌破15%立即剔除）。
    - 防御型组合每半年复查一次，允许ROE波动±2%。
    - 激进型组合每月跟踪，若营收增速连续2季度<15%则剔除。


## 五、防御型利润表和激进型利润表的编制方法

### 1. 防御型利润表

**编制步骤：**

1. 调整项目：

- 资本支出：将现金流量表中“购建固定资产、无形资产和其他长期资产支付的现金”全额计入当期费用（而非资本化）
- 营运资本投资：计算净营运资本变动（`Δ应收账款+Δ存货−Δ应付账款`），并计入费用
- 投资收益：剔除非经营性收益（如股权投资收益）

2. 计算公式：

    `防御型利润=净利润−资本支出−Δ净营运资本−投资收益`
    
或简化为：

    `防御型利润=经营性现金流−资本性支出`
    
（数据来源：现金流量表“经营活动现金流净额”和“投资活动现金流-购建固定资产”）

3. 实操示例（A股）：

- 从现金流量表提取“经营活动现金流净额”和“购建固定资产支出”。
- 从资产负债表计算净营运资本变动（应收账款、存货、应付账款期末与期初差额）。
- 若某公司净利润为10亿，资本支出5亿，净营运资本增加2亿，则防御型利润为10−5−2=3亿。

### 2. 激进型利润表

**编制步骤：**

1. 调整项目：

- 无形资产资本化：将利润表中费用化的研发支出、广告费等调整为资产，按受益期摊销（如研发费用分5年摊销）
- 权益资本成本：计算股东权益的机会成本（如按8%的权益成本率：权益成本=股东权益×8%）

2. 计算公式：

    `激进型利润=息税前利润（EBIT）+资本化的无形资产摊销−权益资本成本`
    
或简化为：

    `激进型资本收益率= 总资产/EBIT `
    
（数据来源：利润表“研发费用”、资产负债表“股东权益”）

3. 实操示例（A股）：

- 从利润表提取EBIT和研发费用（如5亿），假设分5年摊销，则每年资本化1亿。
- 计算权益成本（股东权益100亿×8%=8亿）。
- 激进型利润为20+1−8=13亿。

### 3. A股财务报表中的具体应用
1. 数据来源：

- 防御型利润表：
    - 资本支出：现金流量表“购建固定资产、无形资产和其他长期资产支付的现金”
    - 净营运资本：资产负债表“应收账款”“存货”“应付账款”变动值
- 激进型利润表：
    - 研发费用：利润表“研发费用”科目（需自行调整资本化）
    - 权益成本：资产负债表“所有者权益合计”×权益成本率

2. 关键指标验证：

- 防御型：自由现金流（经营性现金流 - 资本支出）需持续为正
- 激进型：ROE ≥15%、毛利率 ≥40%、息税前资本收益率 ≥18%

3. 交叉验证工具：

通过盈利能力图（横轴激进型利润、纵轴防御型利润）分析企业所处象限，优先选择右上象限（双正）且呈阶梯递增趋势的公司

### 4. 注意事项

1. 会计准则差异：中国研发费用需全部费用化（科创板允许部分资本化），编制激进型利润表需手动调整
2. 风险信号：
    - 防御型：资本支出/折旧 > 2倍行业均值（可能虚增利润）
    - 激进型：客户集中度（CR5）>50%或市销率（PS）>行业2倍
