每个案例都按她的“六步法”走一遍。你会发现，从“模糊的宏大叙事”到“具体的买入标的”之间，她干的那些活儿，跟我们以为的"AI炒股"完全不是一回事。


## 案例 1：AXTI（InP 衬底）—— 一战封神

这是让她在 Reddit WallStreetBets 一战成名、后被封号的案例，也是“紫苏叶理论”最典型的应用。后来她被封号，据说是因为帖子太硬核，怀疑是"机构马甲" 😂。

### 📌 第一步：确认超级趋势

她 **没有** 问“AI 概念股买什么”——这是散户最常犯的错。

她先锁定的是一个 **物理级的确定性**：

> 🏗️ 全球 AI 数据中心正在疯狂扩建，GPU 算力需求指数级增长。这不是"可能发生"的猜测，而是巨头们已经在 Capex 里真金白银砸下去的现实。

然后她拉了一条逻辑链，注意看这条链怎么一步步往下钻的：

```
AI 训练和推理
  → 需要海量 GPU
    → GPU 之间需要高速互联
      → 互联需要光模块
        → 光模块需要激光器
          → 激光器需要 InP 衬底
```

**关键问题来了：InP 衬底从哪来？** 🤔  

大多数人研究到"光模块"就停了——中际旭创、新易盛，研报满天飞，闭眼买就行了。但 Serenity 继续往下挖了 **三层**。

### 📌 第二步：物理级拆产业链

她把“AI 算力扩张”拆成了这么一条 **物理供应链**：

```
AI 需求
  └→ GPU 集群（英伟达）
      └→ 高速互联（NVLink / InfiniBand）
          └→ 光模块（800G / 1.6T）
              └→ 激光器（DFB / VCSEL / 调制器）
                  └→ InP（磷化铟）衬底  ⬅️ 挖到这里，大多数人已经懵了
                      └→ 高纯 InP 多晶原料
                          └→ VGF 晶体生长炉  ⬅️ 她挖到了这里
```

💡 注意：她没有停在“光模块”这一层，而是继续往下拆了三层，直到“InP 衬底”和“InP 多晶原料”。

### 📌 第三步：锁定瓶颈环节

沿着这条链，逐层问了三个问题：**几家能做？扩产要多久？卡不卡脖子？**

| 层级 | 供应商数量 | 扩产周期 | 是不是瓶颈？ |
| :--- | :--- | :--- | :--- |
| GPU | 英伟达垄断，但产能在扩 | 6-12个月 | 紧，但不是最卡的 |
| 光模块 | 中际旭创、新易盛等10+家 | 6-9个月 | 卷得飞起，难出超额收益 |
| 激光器 | Lumentum、Coherent 等5-6家 | 12-18个月 | 有壁垒，但还有得选 |
| **InP 衬底** | **AXTI + 日本住友，实质2家** | **18-24个月，良率爬坡慢** | **✅ 严重瓶颈** |
| InP 多晶原料 | AXTI 自供 + 极少数来源 | 24-36个月 | ✅ 更上游的瓶颈 |

她当时的结论，翻译成大白话就是：

> 🎯 **InP 衬底就是整条链上的“紫苏叶”**——  
> 1. 没有它，激光器出不来；没有激光器，光模块就是废铁；没有光模块，GPU 互联就跑不起来。  
> 2. 全球能批量供应高质量 InP 衬底的，就两家。而 AXTI 是唯一一个“纯玩这个”的上市公司。  
> 3. 扩产极难：VGF 法生长 InP 晶体，良率爬坡要 18-24 个月。**这不是砸钱就能速成的活儿。**

### 📌 第四步：找上市公司 & 候选标的

| 公司 | 市值（当时） | 业务纯度 | 竞争格局 |
| :--- | :--- | :--- | :--- |
| **AXTI** | ~3亿美元 💰 | InP衬底占营收70%+ | 全球仅2-3家能批量供 |
| 住友电工 | ~200亿美元 | InP 只是边角料部门 | 弹性小 |

选 AXTI 的理由，四个字就能概括：**又纯又小**。  

1. **纯 play**：InP 衬底是其核心业务，AI 光模块需求爆发直接转化为营收  
2. **小市值**：3亿美元，机构覆盖极少，估值折价严重  
3. **技术垄断**：VGF 法长晶体有专利和工艺壁垒，不是谁都能抄  
4. **客户实锤**：已经进了 Lumentum、Coherent 等大厂的供应链  

### 📌 第五步：交叉验证 & 风控

**这里是她和"故事型投资者"最大的区别。** 她不是听完逻辑就 all in，而是搞了一套“多源拼图”：

| 验证维度 | 她具体干了啥 |
| :--- | :--- |
| 技术可行性 | 翻 InP 衬底的论文和专利，确认 VGF 法的工艺壁垒到底多高 |
| 产能真实性 | 扒 AXTI 历次财报电话会，跟踪扩产计划和良率数据 |
| 客户验证 | 从光模块大厂（Lumentum 等）的供应商名单里反查 AXTI 的地位 |
| 替代风险 | 研究硅光、CPO 等替代路线，判断 InP 在 800G/1.6T 时代是否仍为主流 |
| 估值安全 | 对比 AXTI 历史估值、同行业估值，判断当前是否被低估 |
| 地缘风险 | AXTI 是美国公司，InP 多晶原料部分依赖进口，评估供应链风险 |

她的 **“反方表”** 核心问题：  

- ❓ GaAs 衬底能不能替代 InP？→ 在高速激光器场景短期不可替代  
- ❓ 住友会大幅扩产砸价吗？→ 住友 InP 只是边缘业务，扩产意愿低  
- ❓ AXTI 会不会稀释股权？→ 关注其增发历史和管理层信誉  

**说白了就是：在买入之前，先把自己当成做空的人，把能想到的坑全走一遍。**

### 📌 第六步：仓位与持有

- 在 AXTI ~3-5 美元区间慢慢建仓，仓位较重  
- 持有期间紧盯：每季财报、电话会、行业展会动态  
- 股价涨到 10+ 美元后，根据估值和瓶颈缓解程度逐步减仓  
- 最终收益：数倍回报（具体倍数各来源说法不一，但确实是赚麻了）

## 案例 2：SIVE（CPO 激光器）—— 从光模块再往下一层

这是她在 X（Twitter）上公开聊过的案例。展示了她如何在“光模块”这个已经热门的赛道里，再找到更隐蔽的瓶颈。

### 📌 第一步：确认超级趋势

> AI 数据中心从 800G 向 1.6T 演进，功耗和密度成了核心矛盾。CPO（共封装光学）成为必然路径。

### 📌 第二步：物理级拆产业链

```
CPO 趋势
  └→ CPO 光引擎
      └→ 外置激光器（ELS）  ⬅️ 为什么外置？因为太热了，放芯片旁边会烤坏
          └→ 高功率 DFB / DBR 激光器芯片
              └→ InP 外延片
                  └→ MBE / MOCVD 设备
```

### 📌 第三步：锁定瓶颈

| 层级 | 瓶颈程度 | 为什么卡？ |
| :--- | :--- | :--- |
| CPO 光引擎 | 🟡 中 | 多家在研，方案还没定型 |
| 外置激光器 ELS | 🟠 高 | 功率、可靠性要求极高，供应商极少 |
| **高功率 DFB 激光器芯片** | **🔴 极高** | **需要特种 InP 外延，全球能做的 ≤3 家** |
| MBE 设备 | 🟠 高 | 供应商极少，交期长 |

### 📌 第四步：找标的

关注到 **Siverse（SIVE）**：  

1. **主业**：高功率 InP 激光器芯片，专供 CPO 外置激光源  
2. **市值**：极小，~1-2亿美元  
3. **客户**：已经在给多家 CPO 方案商送样  
4. **纯 play**：收入几乎全来自 InP 激光器  

### 📌 第五步：交叉验证

| 维度 | 她验证了啥 |
| :--- | :--- |
| 技术路线 | CPO 一定用外置激光器吗？→ 短期这是主流方案 |
| 客户进展 | 是否进入主流 CPO 方案商供应链？→ 有送样和测试信息 |
| 竞争格局 | 还有谁在做？→ Lumentum 有，但 SIVE 是纯 play + 小市值，弹性大 |
| 稀释风险 | 小市值公司经常增发稀释 → 查其融资历史和现金状况 |

### 📌 第六步：仓位与持有

- **小仓位试探**——因为 CPO 路线尚未完全定型  
- 跟踪 CPO 标准进展和客户认证节奏  
- **条件触发式操作**：CPO 大规模商用确认 → 加仓；技术路线生变 → 果断止损

## 案例 3：绿的谐波

这是她在 A 股市场的应用，说明这套方法论 **不挑市场**，核心逻辑在哪都能跑通。

### 📌 第一步：确认超级趋势

> 🤖 人形机器人从概念走向量产，谐波减速器是关节的核心部件——可以把它理解为机器人的“膝盖软骨”，没它动不了。

### 📌 第二步：物理级拆产业链

```
人形机器人
  └→ 关节模组
      └→ 谐波减速器
          └→ 柔性齿轮（柔轮）  ⬅️ 核心中的核心
              └→ 特种钢材 + 热处理工艺
                  └→ 精密加工设备
```

### 📌 第三步：锁定瓶颈

| 层级 | 瓶颈程度 | 为什么卡？ |
| :--- | :--- | :--- |
| 关节模组 | 🟡 中 | 集成商不少 |
| **谐波减速器** | **🔴 高** | **全球就哈默纳科 + 绿的谐波等几家能做，良率爬坡极慢** |
| 柔轮 | **🔴🔴 极高** | 材料配方 + 热处理是核心机密，良率感人 |
| 精密加工设备 | 🟠 高 | 依赖日本进口，交期长 |

### 📌 第四步：找标的

| 公司 | 市值 | 纯度 | 江湖地位 |
| :--- | :--- | :--- | :--- |
| **绿的谐波** | ~200亿人民币 | 谐波减速器占营收80%+ | 国内龙头，全球第二 |
| 哈默纳科 | ~2000亿日元 | 纯 play | 全球老大，但估值已高 |

选绿的谐波的理由：  

1. 谐波减速器是机器人关节的 **必需品**，没有平替  
2. 国内唯一能批量供货的选手  
3. 柔轮良率提升极慢，扩产周期 2-3 年，供给弹性很低  
4. 相比哈默纳科，估值还有空间  

### 📌 第五步：交叉验证

| 维度 | 验证内容 |
| :--- | :--- |
| 技术壁垒 | 查专利、论文，确认柔轮材料和热处理的工艺壁垒 |
| 客户验证 | 追踪特斯拉 Optimus、国内机器人厂商的供应商动态 |
| 替代风险 | 行星减速器能替代吗？→ 人形机器人场景下，谐波更优 |
| 估值 | 和哈默纳科对标，看绿的谐波是不是还有折价空间 |

### 📌 第六步：持有与风控

- 她在 X 上公开提到绿的谐波后，股价快速上涨  
- 但她也提示了风险：A股散户跟风严重，波动大  
- 核心盯两个变量：谐波减速器国产替代进度 + 机器人量产节奏

## 🤖 一个实战Prompt示例

假设想用她的方法分析一个新赛道（比如“固态电池”），可以这样给 AI 下指令：

### 💬 Prompt 1：拆产业链

```
我需要你帮我把“固态电池产业化”这条供应链，从终端产品一直拆到最上游的原材料和设备。

要求：
1. 逐层拆解，不要跳步
2. 每一层标注：该环节的核心产品是什么？全球主要供应商有哪些？是否上市公司？
3. 特别标注：哪些环节的供应商 ≤3 家？哪些环节的扩产周期超过 18 个月？
4. 用表格形式输出
```

### 💬 Prompt 2：找瓶颈

```
基于上面的产业链拆解，请帮我分析：

1. 如果固态电池产能未来3年扩张10倍，哪3个环节最可能成为供给瓶颈？
2. 每个瓶颈环节，请评估：
   - 物理不可替代性（1-5分）
   - 供给刚性/扩产难度（1-5分）
   - 认知稀缺/市场关注度（1-5分）
3. 综合评分最高的瓶颈是哪个？对应的上市公司有哪些？
```

### 💬 Prompt 3：做反方表

```
假设我打算投资 [某瓶颈环节的某公司]，请扮演最挑剔的魔鬼代言人，帮我列出所有可能出错的情况：

1. 技术路径风险：这个瓶颈会不会被新技术绕开？
2. 竞争风险：有没有潜在的进入者？
3. 估值风险：当前市值是否已透支未来3年预期？
4. 管理层风险：有没有治理问题、稀释风险？
5. 地缘/政策风险：有没有制裁、关税、环保等隐患？

请尽可能具体，不要泛泛而谈。
```

### 💬 Prompt 4：证据分级

```
对于 [某公司] 的投资逻辑，请帮我把我目前掌握的信息按可靠性分级：

Level 1 - 一手源事实（财报、公告、专利、官方文件）
Level 2 - 管理层声称（电话会、访谈）
Level 3 - 行业推断（基于公开数据的推算）
Level 4 - 市场传言

我的核心买入逻辑中，有多少是 Level 1-2 支撑的？如果是 Level 3-4 为主，我需要补充什么证据？
```

## 总结：她的方法本质上是一套“认知套利系统”

把三个案例摆在一起看，会发现她的方法有一个非常清晰的内核：

| 维度 | 她的做法 | 大多数人的做法 |
| :--- | :--- | :--- |
| **起点** | 从物理供应链出发，找刚性约束 | 从概念和故事出发，追热点 |
| **选股** | 买“卡脖子的小节点” | 买“最显眼的大龙头” |
| **AI 用法** | 让 AI 拆链、整理、挑刺 | 让 AI 直接推荐股票代码 |
| **验证** | 多源拼图 + 反方表 | 看一篇研报就信 |
| **持有** | 深度研究 → 高信念 → 拿得住 | 追涨杀跌，频繁换手 |
| **风控** | 事前列出“什么情况下我会认错” | 亏了才想"怎么办" |

**说白了，真正值得抄的不是她的股票代码，而是这套完整的流程：**  

> 📐 从物理约束出发 → 🤖 用 AI 放大研究能力 → 🛡️ 用反方表控制风险  

这套方法可以用在任何行业——不一定非得是 AI 或半导体。只要存在 **“高景气需求 + 刚性供给约束”** 的地方，就藏着紫苏叶 🍃。
